When I was still a Web Developer, I was in-charge of maintaining and enhancing a system that allows public to upload documents into a repository when they are applying to become a Permanent Resident in Singapore. While utilising the system to perform testings, I felt that the user experience could be further improved. Hence, I've decided to take the initiative to improve the experience and make it more user-friendly.

As the system had to undergo a technical refresh, it could use a new look-and-feel. Hence, I did my research and proposed my solution to the team for consideration.

The goal of the project is to move towards a paperless approach, to reduce the need for public to come down physically to submit documents so as to improve the efficiency of e-Services application process.

Proposed Design

Users felt overwhelmened by the "long scrolling" due to the number of documents required for upload. This gave them the impression that the process is very cumbersome. The new design (shown below) aims to categorise documents and make the page more concise giving users more "breathing space".

Here is the list of enhanced features that were not present in the application system:

  • Clickable person name: To allow users to freely navigate and upload selected family members documents
  • Collapsible Panel: To properly segment the different categories of documents so as to only display relevant information to users and hide other segments until activated
  • Document Counter: To let users know how many more documents left to upload
  • File Size Counter: To allow users to be aware of the maximum file size capacity
  • Multiple Actions: To allow users to easily delete, download and view their uploads

The Results

All in all, incentives is greater than friction. Even though users may struggle with the uploading of documents, they will still do it because it is the only way they can become a Singaporean PR. However, we should still design the system to meet the users' emotional expectations so as to give a good impression of Singapore and its' public services.
